# MapleStory ARC Progress Simulator

An interactive web-based tool designed for MapleStory players to estimate their Arcane Power (ARC) growth. This simulator provides accurate date-based projections and supports management for multiple characters.

## 🌟 Key Features

- **Multi-Character Management**: Save and switch between different characters.
- **Accurate Projections**: Calculates symbol growth with a logic that resets weekly gains every Thursday.
- **Total ARC View**: Default view shows your total panel ARC (including base 20 per symbol, Hyper Stats, and Guild Skills).

## 🚀 How to Use

1. Visit the [Live Demo](https://cryptoforge0206.github.io/MapleStory-ARC-Progress-Simulator/).
2. Create a character by entering a name and clicking **Add**.
3. Input your current ARC levels, quantities, and daily/weekly gains.
4. Toggle between **Total ARC** and **Individual Levels** to see your progress.
5. Your data will be automatically saved whenever you make a change.

# 楓之谷 ARC 進度模擬器 (MapleStory ARC Progress Simulator)

這是一個專為《楓之谷》玩家設計的網頁工具，用於估算奧術球 (ARC) 的成長進度。本模擬器提供基於日期的精確預測，並支援多角色管理功能。

## 🌟 核心功能

- **多角色管理**：能儲存並自由切換不同角色的進度。
- **精確進度預測**：內建每週四自動結算每週獲得數量的邏輯，提供最精確的完成日期。
- **總面板數值顯示**：預設顯示總面板 ARC（包含每顆球體基礎值 20、極限屬性以及公會技能）。

## 🚀 如何使用

1. 訪問 [線上展示 (Live Demo)](https://cryptoforge0206.github.io/MapleStory-ARC-Progress-Simulator/)。
2. 輸入角色名稱並點擊 **「新增」** 來建立角色。
3. 輸入您目前的 ARC 等級、數量，以及每日/每週可獲得的數量。
4. 在 **「總面板」** 與 **「個別等級」** 之間切換以查看進度圖表。
5. 每次更改數值時，系統都會自動儲存您的資料。

# メイプルストーリー ARC 進捗シミュレーター

『メイプルストーリー』のプレイヤー向けに設計された、アーケインシンボル (ARC) の成長進捗を推定するためのウェブツールです。正確な日付ベースの予測を提供し、複数キャラクターの管理にも対応しています。

## 🌟 主な機能

- **マルチキャラクター管理**: 複数のキャラクターを保存し、簡単に切り替えることができます。
- **正確な予測**: 毎週木曜日に週次獲得分をリセット・加算するロジックにより、正確な完了日を算出します。
- **合計 ARC 表示**: デフォルト設定で合計パネル ARC（各シンボルの基本値 20、ハイパーステータス、ギルドスキルを含む）を表示します。

## 🚀 使い方

1. [ライブデモ (Live Demo)](https://cryptoforge0206.github.io/MapleStory-ARC-Progress-Simulator/) にアクセスします。
2. キャラクター名を入力し、**「追加」** をクリックして作成します。
3. 現在の ARC レベル、個数、および日次/週次の獲得量を入力します。
4. **「合計 ARC」** と **「各レベル」** を切り替えて進捗グラフを確認します。
5. 数値を変更するたびに、データは自動的に保存されます。
